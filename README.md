## 快速开始

### 首次使用

1. 在 iPhone 上打开 **Watch** 应用
2. 找到 **HealthPet Buddy** 并点击安装到 Apple Watch
3. 在 Apple Watch 上打开 HealthPet Buddy 应用
4. 选择你的宠物伙伴（小兔、小猫等）
5. 允许 HealthKit 权限以同步健康数据

### 权限设置

HealthPet Buddy 需要以下 HealthKit 权限：

| 权限 | 用途 |
|------|------|
| 步数 | 驱动宠物成长 |
| 运动时间 | 影响宠物状态 |
| 心率 | 判断宠物情绪状态 |
| 心率变异性 (HRV) | 检测压力状态 |
| 睡眠分析 | 判断疲劳状态 |
| 静息心率 | 计算相对心率变化 |

**如何开启权限**：
1. 打开 iPhone 上的 **健康** 应用
2. 点击右上角头像 → **App** → **HealthPet Buddy**
3. 开启需要的数据访问权限

## 常见问题

### 🐾 宠物相关

#### 宠物为什么不成长？
- 确认已开启 HealthKit 步数权限
- 检查今日是否有新的步数数据
- 宠物成长基于步数达成率，确保有实际运动
- 免费用户宠物成长到「破壳期」后需要解锁高级版

#### 宠物有哪些状态？
宠物会根据你的健康数据展现 5 种状态：

- 😐 **普通**：默认状态
- 💤 **放松**：心率较低，睡眠充足
- ✨ **兴奋**：心率较高（运动后）
- ⚡ **压力**：HRV 偏低（需要放松）
- 🥱 **疲劳**：睡眠不足 6 小时

#### 如何解锁所有宠物？
点击宠物页面右上角的「解锁全部」按钮，通过应用内购买解锁高级版，即可使用全部 8 种宠物和完整成长阶段。

### 👥 好友系统

#### 如何添加好友？
1. 在 Apple Watch 上打开 HealthPet Buddy
2. 进入「设置」→「好友」
3. 查看你的 6 位邀请码
4. 将邀请码分享给好友
5. 输入好友的邀请码完成配对

#### 好友数据不同步怎么办？
- 确保双方都已连接 iCloud
- 检查网络连接
- 好友数据每日自动同步一次（午夜后）

### 📊 小组件

#### 如何添加小组件？
1. 在 iPhone 主屏幕长按空白处
2. 点击左上角「+」按钮
3. 搜索「HealthPet Buddy」
4. 选择喜欢的小组件样式
5. 点击「添加小组件」

#### 小组件不更新？
- 确保已打开过 HealthPet Buddy 应用
- 检查 iPhone 和 Apple Watch 的数据同步
- 重启 iPhone 后重试

### 💳 购买问题

#### 如何恢复购买？
1. 在 Apple Watch 上打开 HealthPet Buddy
2. 进入购买页面
3. 点击「恢复购买」按钮
4. 使用购买时的 Apple ID 登录

#### 购买后未解锁？
- 确认使用的是购买时的同一 Apple ID
- 检查网络连接
- 重启应用后重试
- 如仍有问题，请联系支持邮箱

## 使用技巧

### 最大化宠物成长
- 保持每日运动习惯
- 连续运动天数会有额外成长加成（最高 +50%）

### 保持宠物健康状态
- 保证充足睡眠（7-8 小时）
- 定期运动降低压力状态
- 关注晨间报告了解昨日健康总结

### 与好友互动
- 添加好友后可以看到对方的宠物
- 互相激励保持运动习惯
- 查看好友的昨日数据和状态分布

## 隐私说明

HealthPet Buddy 尊重并保护你的隐私：

- ✅ 健康数据仅存储在你的设备和 iCloud 私有数据库中
- ✅ 我们不会收集、分析或共享你的个人健康数据
- ✅ 好友系统仅共享宠物信息，不共享具体健康数值
- ✅ 你可以随时在系统设置中撤销权限

详细隐私政策请查看 App Store 页面。
